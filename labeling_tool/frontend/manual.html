系统简介

欢迎使用智能检测系统。本系统是一款集成了深度学习目标检测、图像预处理与拼接、数据标注以及统计分析于一体的综合性平台，旨在为缺陷检测、目标识别等工业应用场景提供端到端的解决方案。

核心功能模块

  • 推理和分析： 提供基于深度学习模型（如YOLO系列）的图像自动化推理与检测。支持图像预处理（亮度、对比度调整等）、透视校正、图像拼接，以及一键式的批量检测。
  • 数据标注： 提供高效的图像边界框（Bounding Box）标注工具。支持快捷键操作、数据管理、标签类别自定义，可用于模型训练数据的快速构建与校对。
  • 统计分析： 对推理结果或标注数据进行多维度的可视化分析，生成直观的图表（饼图、柱状图等），帮助用户快速了解数据分布与缺陷情况。
提示： 建议在使用本系统前，确保已正确配置好GPU环境（CUDA），以获得最佳的推理与处理速度。在系统的"推理和分析"页面右上角可以查看当前GPU状态。

推理和分析

该模块主要用于对输入图像进行自动化模型推理与缺陷检测。您可以添加单张图片或整个文件夹，进行预处理后提交推理任务。

1. 新建推理任务

在界面左侧区域，您可以新建推理任务：

  • 添加数据源： 点击 [添加文件] 或 [扫描文件夹] 导入需要检测的图像（支持jpg, png等格式）。
  • 推理设置： 点击右上角 [推理设置]，可调整置信度阈值（Confidence）和交并比阈值（IoU）等参数。也可以选择使用哪种模型进行推理。
  • 提交任务： 确认文件列表后，点击 [开始推理]。系统将自动在后台进行处理。

2. 图像预览与预处理

在界面中间的预览区域，您可以对选中的图片进行预处理操作，以提升检测效果：

  • 视图模式： 查看当前图像，在下方的 图像预处理 栏可以调整亮度、对比度、曝光等参数。
  • 透视校正： 切换到"透视校正"模式，拖动控制点（四角），校正图像的透视畸变。
  • 图像拼接： 在文件列表中勾选多张图片，点击 [图像拼接]，系统将尝试将多张局部图片拼接成一张全景图。
  • 调整完毕后，可点击 [应用当前设置] 或 [应用至选中项] 以保存预处理参数。

3. 任务管理与结果分析

在界面右侧区域，显示所有历史及当前的推理任务：

  • 您可以实时查看任务的运行状态（排队中、运行中、已完成、错误）。
  • 对于已完成的任务，点击 [分析] 按钮，进入结果详情页，查看模型检测出的目标与缺陷框。
  • 对于失败或中断的任务，可尝试重新运行或删除。

数据标注

数据标注模块用于人工对图像中的目标或缺陷进行边界框（BBox）绘制与类别指定，生成用于模型训练的高质量数据集。

1. 数据集选择与图像浏览

  • 在左侧面板选择需要标注的目录或数据集。
  • 下方会列出该目录下的所有图片。点击图片即可在中央画布中打开。
  • 带有 标记的图片表示已完成标注。

2. 标注操作指南

在中央画布区域：

  • 绘制拉框： 在图像上按住鼠标左键并拖动，即可绘制一个新的边界框。
  • 修改类别： 选中已绘制的框，在右侧属性面板或弹出的下拉框中选择对应的缺陷类别。
  • 调整大小与位置： 选中框后，拖动框的边缘控制点以缩放，或拖动框体进行移动。
  • 平移与缩放画布： 滚动鼠标滚轮可缩放画布；按住 空格键 + 鼠标左键拖动 可平移画布。

3. 快捷键列表

  • 左键拖动 : 绘制边界框 / 移动边界框
  • 滚轮 : 缩放图片
  • 空格 + 拖动 或 右键拖动 : 平移图片画布
  • Delete 或 Backspace : 删除选中的边界框
  • A / D 或 ← / → : 切换上一张 / 下一张图片
注意： 系统会自动保存您的标注结果。无需手动点击保存按钮。

统计分析

统计分析模块提供直观的数据可视化功能，帮助您宏观把握检测或标注数据的整体情况。

1. 查看统计图表

  • 模块加载后，系统会自动聚合当前数据集的缺陷与目标信息。
  • 饼图/环形图： 展示各类缺陷（如划痕、裂纹、异物等）在总体中所占的比例分布。
  • 柱状图： 展示各类别在具体数量上的对比，方便识别高频缺陷类型。

2. 报告导出

您可以根据可视化的图表及详细数据表，导出综合分析报告（如果系统提供导出按钮），以便于汇报或进一步的数据归档。

常见问题 (FAQ)

Q: 系统提示 "CUDA 不可用，正在使用 CPU 模式" 会有什么影响？

A: 这意味着系统无法调用NVIDIA独立显卡进行硬件加速，而只能使用CPU进行计算。这会导致图像拼接、预处理以及深度学习模型推理的速度大幅下降。建议检查显卡驱动及CUDA环境配置是否正确。

Q: 图像拼接失败或效果不好怎么办？

A: 图像拼接依赖于相邻图像之间的重叠区域与特征点。如果重叠过少、图像纹理单一或存在大幅度曝光差异，拼接可能失败。您可以尝试先在"图像预处理"中统一各图的亮度/对比度，然后再进行拼接。

Q: 画布无法拖动或操作卡顿？

A: 请确保浏览器硬件加速已开启。若加载的图片分辨率极高，可能会导致轻微卡顿，请耐心等待渲染。拖动画布时请确保按住了空格键或使用鼠标右键。
